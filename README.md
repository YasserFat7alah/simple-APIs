# 🔐 Authentication API

A simple RESTful Authentication API that supports user registration and login.

---

## 🚀 Tech Stack

- Node.js
- Express
- PostgreSQL (NEON)
- Drizzle-orm
- Bcrypt for password hashing

---

```bash
git clone https://github.com/YasserFat7alah/simple-APIs/tree/authentication-api
